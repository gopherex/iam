# RFC 7515 — JSON Web Signature (JWS)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7515 · Conf: MUST

## Назначение
Подпись JWT. Header `alg`, `kid` (ключ из JWKS).

## Где в API
- §22 подпись токенов, §33 signing keys

## Conformance
- `kid` обязателен — для ротации (§33). Verifier выбирает ключ по kid из JWKS.

## Gotchas
- Поддержать несколько активных ключей при ротации (старый валиден до истечения токенов).
