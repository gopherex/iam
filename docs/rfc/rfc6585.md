# RFC 6585 — Additional HTTP Status Codes

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc6585 · Conf: MUST

## Назначение
`429 Too Many Requests`, `428 Precondition Required`, `431`, `511`.

## Где в API
- §37 rate-limit, §34 rate-limits config

## Conformance
- 429 при превышении лимита + `Retry-After`. error code `rate_limited` (§0).

## Gotchas
- 429 на auth-ручках — не раскрывать существование аккаунта в сообщении.
