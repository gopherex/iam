# RFC 8037 — CFRG Curves (EdDSA / Ed25519) for JOSE

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc8037 · Conf: SHOULD

## Назначение
alg `EdDSA` (Ed25519) для JWS — быстрая и компактная подпись.

## Где в API
- §33 signing keys (опция alg)

## Gotchas
- Проверить поддержку EdDSA в SDK-verifier'ах клиентов; не все стеки умеют. Держать ES256/RS256 как fallback в JWKS.
