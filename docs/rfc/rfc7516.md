# RFC 7516 — JSON Web Encryption (JWE)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7516 · Conf: OPT

## Назначение
Шифрование payload токена (конфиденциальность claim'ов).

## Где в API
- §22 (опц. зашифрованные id_token), §38 SSO (encrypted assertions)

## Gotchas
- Нужен только если claim'ы чувствительны и токен виден третьим. Обычно подписи (JWS) достаточно.
