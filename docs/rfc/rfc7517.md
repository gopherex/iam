# RFC 7517 — JSON Web Key (JWK / JWKS)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7517 · Conf: MUST

## Назначение
Публикация публичных ключей: `/.well-known/jwks.json` со списком JWK (kid, kty, alg, use).

## Где в API
- §2 `/.well-known/jwks.json`, §33 admin jwks list/rotate/activate

## Conformance
- Только публичные ключи. Включать текущий + предыдущие (grace period ротации).
- Cache-Control разумный; verifier рефрешит по kid-miss.

## Gotchas
- Удалять старый ключ только после истечения всех выданных им токенов.
