# RFC 7643 — SCIM Core Schema

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7643 · Conf: MUST

## Назначение
Схемы ресурсов: User, Group, EnterpriseUser; атрибуты, мета, `externalId`.

## Где в API
- §41 Users/Groups, ServiceProviderConfig/Schemas/ResourceTypes

## Conformance
- Поддержать core User (userName, name, emails, active), Group (displayName, members).
- `externalId` — связка с id в IdP.

## Gotchas
- `active:false` = деактивация (не удаление) — маппить на suspend/disable юзера.
