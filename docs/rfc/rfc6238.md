# RFC 6238 — TOTP (Time-Based One-Time Password)

- Status: Informational · https://www.rfc-editor.org/rfc/rfc6238 · Conf: MUST

## Назначение
Authenticator-app коды: HMAC от текущего time-step + секрет.

## Где в API
- §10 `/mfa/totp/enroll`, `/mfa/totp/verify`

## Conformance
- step 30s, 6 цифр, SHA-1 (совместимость Google/Authy). otpauth:// URI + QR.
- Окно ±1 step (clock drift). Один код — одно использование (anti-replay).

## Gotchas
- Секрет — base32. Хранить зашифрованным. enroll подтверждать verify перед активацией фактора.
