# RFC 9101 — JWT-Secured Authorization Request (JAR)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc9101 · Conf: OPT

## Назначение
Параметры authorization-запроса в подписанном JWT (`request`/`request_uri`) — целостность запроса.

## Где в API
- §23 `/oauth2/authorize`

## Gotchas
- Пересекается с PAR (9126); FAPI-профиль требует одного из них. Для обычных клиентов опционально.
