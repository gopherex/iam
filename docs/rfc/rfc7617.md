# RFC 7617 — HTTP Basic Authentication Scheme

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7617 · Conf: MUST

## Назначение
`Authorization: Basic base64(client_id:client_secret)` — метод `client_secret_basic` для OAuth token endpoint.

## Где в API
- §23 `/oauth2/token` client auth, §25 service account secret

## Conformance
- Поддержать `client_secret_basic` (header) и `client_secret_post` (body). private_key_jwt (7523) — предпочтительнее.

## Gotchas
- Не путать с user Basic auth — здесь только client credentials. Только по TLS.
