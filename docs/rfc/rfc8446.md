# RFC 8446 — TLS 1.3

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc8446 · Conf: MUST

## Назначение
Транспортное шифрование. TLS 1.3 (минимум 1.2).

## Где в API
- Весь трафик

## Conformance
- TLS 1.3 предпочтительно, 1.2 минимум. Отключить <1.2.
- mTLS (8705) — если включён cert-bound.

## Gotchas
- Терминация TLS на gateway → прокидывать `X-Forwarded-Proto`, не доверять http внутри.
