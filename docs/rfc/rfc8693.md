# RFC 8693 — OAuth 2.0 Token Exchange

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc8693 · Conf: SHOULD

## Назначение
grant `urn:ietf:params:oauth:grant-type:token-exchange`: обмен/делегирование токенов (impersonation, downscoping, S2S по цепочке).

## Где в API
- §22 token runtime, §25 service accounts (delegation между сервисами)

## Conformance
- Параметры `subject_token`, `actor_token`, `requested_token_type`, `audience`, `scope`.
- Поддержать downscope (выдать токен с меньшим scope).

## Gotchas
- Не путать с §3 `token/exchange` (это обмен auth_code, внутренний). RFC 8693 — для S2S делегирования.
- impersonation требует authz-проверки актора.
