# RFC 7644 — SCIM Protocol

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7644 · Conf: MUST

## Назначение
REST-протокол SCIM: CRUD Users/Groups, PATCH (PatchOp), filtering, pagination, bulk, `/ServiceProviderConfig`.

## Где в API
- §41 (все Users/Groups ручки), §42 SCIM tokens

## Conformance
- PATCH с `Operations[{op:add/remove/replace, path, value}]`.
- filter (`userName eq "..."`), `startIndex`/`count` пагинация, `ListResponse`.
- Bearer SCIM-token per org (§42), scoped к org_id в URL.

## Gotchas
- Идемпотентность PUT vs PATCH. Okta/AzureAD капризны к точному формату ошибок (`scimType`).
- DELETE часто = deprovision; некоторые IdP только PATCH active:false.
