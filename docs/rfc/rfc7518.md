# RFC 7518 — JSON Web Algorithms (JWA)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc7518 · Conf: MUST

## Назначение
Реестр алгоритмов для JWS/JWE/JWK: RS256, ES256, PS256, HS256, ...

## Где в API
- §33 token-profiles / signing alg

## Conformance
- Default asymmetric: `RS256` или `ES256` (verifier-side JWKS). HS* — только для внутренних, не для публичной верификации.

## Gotchas
- Не использовать HS256 для токенов, верифицируемых внешними сервисами (shared secret утечёт).
