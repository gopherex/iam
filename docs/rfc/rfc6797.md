# RFC 6797 — HTTP Strict Transport Security (HSTS)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc6797 · Conf: SHOULD

## Назначение
`Strict-Transport-Security` — форсит HTTPS, защита от SSL-strip.

## Где в API
- Транспорт (все ответы)

## Conformance
- `max-age>=31536000; includeSubDomains`. Рассмотреть preload.

## Gotchas
- Auth-домен — кандидат на preload-list. includeSubDomains осторожно (поддомены).
