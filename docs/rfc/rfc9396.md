# RFC 9396 — Rich Authorization Requests (RAR)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc9396 · Conf: OPT

## Назначение
`authorization_details` — структурированные fine-grained разрешения вместо плоских scope (напр. «перевод 100 EUR на счёт X»).

## Где в API
- §20 AuthZ runtime, §23 authorize/consent

## Gotchas
- Мощно для transactional consent; для RBAC/обычных API избыточно. Связать с §23a consent.
