# RFC 5322 — Internet Message Format (Email addresses)

- Status: Draft Standard · https://www.rfc-editor.org/rfc/rfc5322 · Conf: MUST

## Назначение
Синтаксис email-адреса (+ RFC 6531 для EAI/Unicode).

## Где в API
- email-валидация: signup, email verify/change, invitations

## Conformance
- Валидировать синтаксис + нормализовать (lowercase домена). Подтверждение владения — через verify (§4), не доверять синтаксису.

## Gotchas
- Не over-валидировать regex'ом (ломает валидные адреса). Реальная проверка = отправка кода.
- Нормализация для uniqueness (gmail dots/plus) — политика, документировать.
