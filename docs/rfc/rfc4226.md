# RFC 4226 — HOTP (HMAC-Based One-Time Password)

- Status: Informational · https://www.rfc-editor.org/rfc/rfc4226 · Conf: MUST (база TOTP)

## Назначение
Counter-based OTP; основа TOTP (6238). Также модель для счётчиков/lookahead.

## Где в API
- §10 MFA (фундамент TOTP)

## Gotchas
- Чистый HOTP редко нужен напрямую; важен как база. Recovery codes — отдельный механизм (не HOTP), просто одноразовые секреты.
