# RFC 9562 — Universally Unique Identifiers (UUID)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc9562 · Conf: SHOULD (заменяет 4122)

## Назначение
UUID, включая UUIDv7 (time-ordered) — удобно для БД-индексов.

## Где в API
- id всех ресурсов (или префиксные ULID-стиль `user_`, `org_`)

## Conformance
- Внутренние PK: UUIDv7 (сортируемость). Внешние id могут быть префиксными (`app_xxx`).

## Gotchas
- Не light на UUIDv4 для hot-PK (фрагментация индекса). v7 лучше.
