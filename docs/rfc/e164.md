# E.164 — International Phone Number Format (ITU-T)

- ITU-T Recommendation E.164 · https://www.itu.int/rec/T-REC-E.164 · Conf: MUST

## Назначение
Канонический формат телефона: `+<country><number>`, до 15 цифр.

## Где в API
- phone-валидация: signup, phone verify/change (§5), SMS MFA (§10)

## Conformance
- Хранить и сравнивать в E.164 (`+14155551234`). Нормализовать ввод (libphonenumber).

## Gotchas
- Uniqueness/верификация только по E.164. Подтверждение — SMS-код (§5), синтаксиса мало.
