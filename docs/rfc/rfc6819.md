# RFC 6819 — OAuth 2.0 Threat Model and Security Considerations

- Status: Informational · https://www.rfc-editor.org/rfc/rfc6819 · Conf: SHOULD

## Назначение
Каталог угроз OAuth и контрмер. Историческая база, дополнена 9700.

## Где в API
- §37 risk/abuse, §23 OAuth

## Gotchas
- Для реализации опираться на 9700 (актуальнее); 6819 — справочник угроз (CSRF, token leakage, redirect, clickjacking).
