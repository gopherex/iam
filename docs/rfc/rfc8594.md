# RFC 8594 — The Sunset HTTP Header Field

- Status: Informational · https://www.rfc-editor.org/rfc/rfc8594 · Conf: SHOULD

## Назначение
`Sunset` header — дата вывода ресурса/ручки из эксплуатации. Парный `Deprecation` header (draft) — что ручка устарела.

## Где в API
- API versioning (idea.md §versioning / spec I.1): устаревающие ручки/версии

## Conformance
- `Sunset: <HTTP-date>` на ручках с планом удаления; `Deprecation: <date|true>` при депрекейте.
- Дублировать в `/config/public` + docs, чтобы SDK знал заранее.

## Gotchas
- Deprecation header — отдельный IETF draft (draft-ietf-httpapi-deprecation-header), не финальный RFC; зафиксировать синтаксис.
