# RFC 3339 — Date and Time on the Internet (Timestamps)

- Status: Proposed Standard · https://www.rfc-editor.org/rfc/rfc3339 · Conf: MUST

## Назначение
ISO-8601-профиль для timestamp: `2026-06-04T12:00:00Z`.

## Где в API
- Все `*_at`, `expires_at`, `time` поля

## Conformance
- UTC с `Z`. Единый формат во всех ответах.

## Gotchas
- JWT `exp/iat/nbf` — это NumericDate (Unix seconds, RFC 7519), НЕ RFC 3339. Не смешивать.
